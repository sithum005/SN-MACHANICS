USER GUIDE: How to update images on your website

1.  **Replacing the Banner Image**:
    - Go to `assets/img/hero/` inside your main folder.
    - Replace `banner.svg` with your own photo.
    - IMPORTANT: If your photo is a `.jpg` or `.png` file (e.g., `banner.jpg`), you must open `index.html` in Notepad or Code editor and change `src="assets/img/hero/banner.svg"` to `src="assets/img/hero/banner.jpg"`.

2.  **Replacing Machine Photos**:
    - Go to `assets/img/machines/`.
    - Place your photos here (e.g., `singer-sl.jpg`, `zoje-industrial.jpg`).
    - Update `index.html` and `services.html` to point to your new file names (change `.svg` to `.jpg` or `.png` in the `<img>` tags).

3.  **Replacing Gallery Photos**:
    - Go to `assets/img/gallery/`.
    - Place your photos here (e.g., `1.jpg`, `2.jpg`, etc.).
    - Update `gallery.html` to use your new file names.

4.  **Editing Text**:
    - Open any `.html` file (index, services, gallery, contact) using Notepad or VS Code.
    - Carefully change the text between the `>` and `<` brackets.
    - Example: Change `<h1...>Expert Repair</h1>` to `<h1...>Your Text Here</h1>`.

Enjoy your new website! - SN Machinics
